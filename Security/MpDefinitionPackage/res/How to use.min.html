How to use

This app extracts antimalware definitions onto your computer. Provided that Microsoft Defender Antivirus is present and enabled, within two minutes, it detects the definitions, verifies their integrity, and installs them – deleting the extracted files in process.

This app also starts Command Prompt and has it navigate to the folder in which the app has extracted the definitions. You can use the Command Prompt window (and the DIR command) to verify that the extracted files gone. Or you can close it if you don't want to.
